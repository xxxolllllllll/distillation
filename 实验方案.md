- **2 异构蒸馏方法**

  - **2.1 异构网络构建与层级对应关系**

  - 为充分利用视觉大模型的表征能力，本文构建了由教师模型与学生模型组成的异构蒸馏框架。教师模型采用 facebook/dinov3-vitb16-pretrain-lvd1689m，学生模型采用 YOLOv11-m。训练与推理阶段输入图像统一为 1024×1024，并采用保持纵横比的缩放与 padding 预处理方式，以减少几何变形对缺陷形态的影响。在层级监督设计上，学生模型输出三层特征 P3/P4/P5，其空间分辨率分别为 128×128、64×64、32×32。教师模型选取第 3、4、7、8、11、12 个编码器块输出作为蒸馏特征，并建立如下对应关系：第 3/4 层对应 P3，第 7/8 层对应 P4，第 11/12 层对应 P5。该映射使浅层纹理细节、中层结构信息与深层语义信息能够在跨架构条件下实现分层对齐。

  - **2.2 多维特征对齐机制**

  - 针对 ViT 与 CNN 在特征表示上的异构性，本文从空间尺度与通道维度两个方面进行联合对齐。

  - （1）空间尺度对齐：以学生特征图分辨率为目标尺度，采用双线性插值对教师特征进行重采样。其中，第 3/4 层教师特征对齐至 128×128，第 7/8 层对齐至 64×64，第 11/12 层对齐至 32×32。双线性插值过程遵循标准像素映射关系：$$( [ x = p\cdot \frac{H_{src}}{H_{dst}},\quad y = q\cdot \frac{W_{src}}{W_{dst}} ] )$$并由相邻四个整数像素点加权求得目标像素值。

  - （2）通道维度对齐：在学生侧分别对 P3/P4/P5 追加 1×1 卷积，将通道数映射至教师嵌入维度 (C_t=768)。具体为：
  P3: 128×128×192 -> 128×128×768，
  P4: 64×64×384 -> 64×64×768，
  P5: 32×32×512 -> 32×32×768。
  该步骤保证教师与学生特征在通道语义空间中的一致性，为后续蒸馏损失计算提供可比基础。

  - **2.3 自适应融合蒸馏与损失函数**
  为增强教师监督信息的层级表达能力，本文对教师相邻层特征执行自适应加权融合。对于每组特征 (F_s, F_d)（对应 3/4、7/8、11/12），先在通道维拼接，再通过输出通道为 2 的 1×1 卷积生成权重映射 (F_y)，并在每个空间位置进行 softmax 归一化，得到权重 (w_1,w_2)。融合后的教师目标特征定义为：$$( [ F_{target,l}(i,j,k)=w_1(i,j)\cdot F_s(i,j,k)+w_2(i,j)\cdot F_d(i,j,k) ] [ w_1(i,j)=\frac{e^{F_y(i,j,1)}}{e^{F_y(i,j,1)}+e^{F_y(i,j,2)}},\quad w_2(i,j)=\frac{e^{F_y(i,j,2)}}{e^{F_y(i,j,1)}+e^{F_y(i,j,2)}} ] [ w_1(i,j)+w_2(i,j)=1 ])$$
  
  在损失设计上，采用 SmoothL1 约束学生对齐特征与教师融合特征之间的差异：$$( [ \mathcal{L}{feat}=\sum{l=1}^{3}\lambda_l\cdot \mathcal{L}{feat,l},\quad \mathcal{L}{feat,l}=\text{SmoothL1}(F^{aligned}{student,l},F{target,l}) ] )$$其中层级权重设置为 $$(\lambda_1=0.5,\lambda_2=0.3,\lambda_3=0.2)$$。该权重分配强调浅层细节知识迁移，以提升裂缝、孔洞等微小缺陷区域的识别能力与边界表达精度。


- **3.1 数据集与标注**
  - **3.1.1 数据预处理与切片** : 暂时确定的整套数据预处理与切片流程（针对语义分割 + 小目标）分成四块：原图级预处理 → 大场景 patch → 小目标 ROI patch→ ROI 缩放 + padding，所有操作对图像和像素级掩膜同步进行。
  - (1). 原图级预处理对原始高分辨率图像及其语义分割掩膜：
  按长宽比不变缩放，使长边不超过某上限（如 3000 或 2048），得到统一尺度的“预处理图像”与“预处理 mask”。  
  此处只是把过大的图整体缩小，方便后续裁剪，不强行压成方形尺寸。  
  - (2). 大场景滑动窗口 patch（偏向腐朽/整体结构）
  在预处理图像上采用固定尺寸窗口（如 1024×1024），规则滑动裁剪：  
  步长可以取 512 或 800（覆盖大致完整即可）。  
  在对应的预处理 mask 上，用相同坐标同步裁剪出 1024×1024 子 mask。  
  对于得到的 (patch_img, patch_mask)：  
  若几乎全是背景或大面积腐朽但轮廓不清，可只保留一小部分，避免训练集被这类样本淹没；  
  其余作为“大场景训练样本”，用于学习整体构件形态和大块腐朽。  
  - (3). 小目标 ROI patch（规则选择判定）
  对每一个小目标（可根据分割 mask 提取连通域或已有框）先计算外接矩形尺寸 $$w \times h$$、中心 $$(c_x,c_y)$$、长宽比 $$r=\max(w,h)/\min(w,h)$$，并按以下条件选择 ROI 规则：  
    - **规则A：细长/非紧凑目标（默认）**  
    当类别为 `crack`，或几何形态满足细长条件（如 $$r>r_{\text{line}}$$，取 5）时，使用自适应矩形 ROI。  
    - **规则B：紧凑极小目标（固定方形）**  
    仅当目标满足“紧凑且极小”条件时，使用固定边长正方形 ROI：  
    $$\max(w,h)\le L_{\text{tiny}} \ \text{且}\ r\le r_{\text{compact}}$$（例如 $$L_{\text{tiny}}=24,\ r_{\text{compact}}=2.5$$）。  
    该规则主要覆盖 `defect` 中孔洞/钉眼形态。  
    - **冲突优先级**  
    若目标同时满足“面积小”和“细长”，优先使用规则A（避免裂缝因面积小被误分到固定方形 ROI）。  
    - **规则A具体处理**  
    若 $$(\max(w,h)/\min(w,h) > r_{\max})$$（如 5），将短边扩展到 $$(\min(w,h)=\max(w,h)/r_{\max})$$，并围绕中心对称扩张；  
    同时施加最小边长约束：$$(w_{\text{ROI}},h_{\text{ROI}} \ge S_{\min})$$（如 128），不足时向两侧补齐。  
    - **规则B具体处理**  
    以目标中心构造固定边长 $$(S_{\text{hole}})$$（如 192）的正方形 ROI。  
    - 最后将 ROI 矩形裁剪到图像边界内，得到最终的 $$((x_{\min}, y_{\min}, x_{\max}, y_{\max}))$$。  
    - (3.2) 从图像与 mask 裁剪 ROI 子图  
    图像：roi_img = I[y_min:y_max, x_min:x_max]  
    掩膜：roi_mask = M[y_min:y_max, x_min:x_max]
- 1. ROI 缩放 + padding 到统一模型输入尺寸:
  - 设模型输入尺寸为 H_in×W_in（如 1024×1024）：记录 ROI 当前尺寸 $$(H_{\text{ROI}}, W_{\text{ROI}})$$。计算缩放比例（同时约束长边和短边不超出输入尺寸）：$$[ \text{scale} = \min\left(\frac{H_{\text{in}}}{H_{\text{ROI}}},\frac{W_{\text{in}}}{W_{\text{ROI}}}\right) ]$$
  - 得到新尺寸：$$(H_{\text{new}} = \text{round}(H_{\text{ROI}} \cdot \text{scale}))$$ $$(W_{\text{new}} = \text{round}(W_{\text{ROI}} \cdot \text{scale}))$$ 对 roi_img、roi_mask 同步缩放到 H_new×W_new（图像用双线性，mask 用最近邻）。
  - 再做 padding：上下左右按差值一分为二填充，把 (H_new, W_new) 补到 (H_in, W_in)。图像 padding 用常数或边缘值，mask padding 用“背景类别”。
  - 后面你只需要把具体的参数（`H_in`、`r_max`、`S_min`、`S_hole`、滑窗步长等）定下来即可细化到代码和论文中。
